AI résumé coach built on a hybrid **GPT-4-mini + GPT-4** pipeline (**–29% cost**, 3× more consistent feedback). A **RAG** system improved skill-extraction accuracy by **15.8%** and project-scope detection by **16.9%**, turning generic output into recruiter-style, role-specific feedback. Includes an **Interview Practice Bot** that transcribes spoken answers and scores pacing, hesitation, and confidence from segment-level timing.
`Next.js 14 · Supabase · GPT-4 · Whisper · RAG · shadcn/ui · TypeScript`

Dynamic multi-agent system that autonomously researches, writes, and edits technology articles through a **SupervisorGraph** orchestration pipeline with LangGraph ReAct nodes, Tavily Search, and GPT-4o.
`Next.js 14 · LangChain · LangGraph · LangSmith · OpenAI · TypeScript`

### 🔍 Job Busters — Ghost-Job Risk Scoring
Prototype scoring system that flags likely "ghost job" postings, producing a 0–100 composite risk index from compensation integrity, posting cadence, metadata consistency, cross-site validation, and NLP-based description analysis.
